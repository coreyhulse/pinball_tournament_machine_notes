#### ROM Version

Do not use v3.20 ROM, it contains a bug that can crash the game. Instead use v3.03 (and Display ROM v3.01).

#### Tech Notes

Monopoly has a (somewhat) secret flipper code.
In attract-mode press: Both 1 time then 7 Left, 1 right, 4 left, 1 right, 4 left, 2 right

Merry Christmas. Courtesy of Greg Dunlap

#### Physical Setup

- Verify Machine Level Machine
- Outlane Posts on "Hard Settings". I.e. most open.
- Make slingshots sensitive
- Install Ear Plug Tilt Hack
- Verify Tilt Sensitivity -- should be sensitive, but only one danger with each hard bump
- Check coin door ball save.
- Check Flipper Strength (adjust #43 setting below if needed)
- Clean/wax playfield if necessary
- Clean glass
- Verify Volume

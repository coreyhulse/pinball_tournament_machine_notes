#### Game Version

- This is the version by Stern, 2025

#### Feature Adjustments

- Enable Virtual Locks
  - This is especially important for the Premium/LE where it may be possible to lock two balls and then shake the game causing the trough to lose count of the balls in play and start the multiball
  
- Consider disabling the gong lock on Premium/LE
  - Make sure you have 48V active
  - Go to Device Tests -> Gong Device Test-> Opened
  - Go to Adjustments -> Machine Settings -> Devices -> Gong Latch Disabled -> Yes

#### Setup Recommendations for Events with Major-level Players

These recommendations come from Colin McAlpine for tournaments that are expecting many top caliber players and need to manage game times.

- Physical changes
  - Install lightning flippers slightly flopped.
    - The feed to the right flipper from both the Gong scoop and the Cave is too easy to control. The lower angle and the lightning flippers prevent an an easy cradle. This also narrows the window for backhanding the Gong. Just make sure that you can still make the center ramp from the right flipper on a clean shot.
    - This is the most important change.
  - Remove the right outlane post.
  - Remove the right inlane guide post rubber.
  - Remove the center drain post.
  - Leave the left outane post in but remove the rubber.
    - If you remove this post it is easier for the ball to make its way into the river feed lane.
  - Remove the rubber from the narrow escape post that's below the left outlane drain area.
  - Set the game around a 7° pitch.
    - Don't make it too steep to avoid letting the feeds from the right VUK be an easy cradle.
- Settings changes
  - Set the tilt debounce to 500ms.
  - Leave Kong MB at its default difficulty.
  - Set Pit MB one or two levels harder.
    - This combined with the changes to the outlane and center drain posts will lead to a lot of drains.
  - Set Skill Fire mode to be lit at 6 Kong letters.
  - Set Building Climb difficulty to Medium (one harder than the default).
  - Set the Cliffhanger left outlane save mode to only award Points.
  - Set the Kong letter spelling difficulty one level harder than default.
    - This makes it so you can't earn KONG letters from switch hits during multiballs.
  - Set the Log Diverter automatic setting to off.
    - The log will still activate automatically during modes like Cross the Chasm.
  - Set the maximum banked log diverter uses to 2.
  - Set the mini-flipper max hold to 3 seconds.

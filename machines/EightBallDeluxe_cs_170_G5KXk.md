#### Fixed ROM

Use the fixed romset to address the issue where players 2 and 4 get extra bonus due to the 8 ball being counted twice. If this ROM is not installed then play as players 1 and 3 only.

Drop Targets should reset with 56k/112k light spotted when completing the DELUXE Stand Ups. Switch 23 = off 8-ball Saucer should collect bonus with lit multiplier. Switch 16 = on

#### Self-test Positions

- #16: 01
  - Sets Specials to be 50 k and left lane extra ball to be 25 k
- #17: 01
  - Disables awards for reaching scoring thresholds

#### Backbox Switches

- SW7 - C and D rollover lane light adjustment (TD choice)
  - ON - Making the C or D lane will unlight both C and D lanes (making it easier to complete ABCD to spot drop targets)
  - OFF - Making the C or D lane will only unlight the lane that was activated
- SW31 and SW32 - Balls per game
  - 5-Ball (for PinGolf): SW31 ON and SW32 OFF
  - 3-Ball: SW31 OFF and SW32 OFF
- SW16 Saucer hole scoring (TD choice)
  - OFF - Saucer scores rack and bonus values without any bonus X. This is the more common choice.
  - ON - Saucer scores rack and bonus values multiplied by bonus X. This can encourage more shots towards drop targets, increasing risk. But a player who achieves a lot of progress earlier can more easily run away with the game.
